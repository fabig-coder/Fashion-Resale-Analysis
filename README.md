#Fashion Resale Market Intelligence
SQL and Python project analyzing secondhand fashion market trends and predicting consumer demand shifts.

# Live App
[Launch App](https://fashion-resale-analysis-neyve3usrvvx26ukuvtsom.streamlit.app/)

# Features
- Real-time brand search interest via Google Trends
- Investment signals (BUY/WATCH/AVOID) based on momentum
- Resale vs Fast Fashion market size comparison
- Search any brand for resale momentum
- Resale value retention calculator
- Stock tracker for The RealReal and ThredUp
- Price comparison tool across RealReal, Poshmark, and eBay

# Tech Stack
Python, Streamlit, Plotly, Google Trends API, Yahoo Finance API, Pandas, Scikit-learn
